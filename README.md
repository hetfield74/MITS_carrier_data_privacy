## Anleitung für das Modul MITS_carrier_data_privacy


## 1. Installation

Systemvoraussetzung: Funktionsfähige modified eCommerce Shopsoftware ab der Version 2.0.0.0 rev 9678

Vor der Installation des Moduls sichern sie bitte komplett ihre aktuelle Shopinstallation (Dateien und Datenbank)!
Für eventuelle Schäden übernehmen wir keine Haftung!
Die Installation und Nutzung des Moduls MITS_carrier_data_privacy erfolgt auf eigene Gefahr!

Die Installation des Modul MITS_carrier_data_privacy ist in der aktuellen Shopversion ziemlich einfach.

    1. Falls der admin-Order des Shops unbenannt wurde, dann entsprechend auch den Ordner admin
       im Verzeichns shoproot des Moduls vor dem Hochladen ebenfalls entsprechend umbenennen!

    2. Im Ordner shoproot finden Sie im Ordner templates die anzupassenden und neuen Templatedateien für die gängigen Standardtemplates.
       Sollte Ihr Template an diesen Stellen unverändert sein, dann können Sie die Dateien einfach überschreiben.
       Sie können die Dateien aber auch z.B. mit dem Programm WinMerge vergleichen und die Änderungen für das Modul so übernehmen.
       Für die manuelle Anpassung der Templatedateien folgen Sie bitte den Anweisungen der beigefügten install_template.txt
       Denken Sie aber in jedem Fall zur Vorsicht auch bitte hier an eine Sicherung der bestehenden Dateien!

    3. Kopieren Sie anschließend einfach alle verbliebenen Ordner und Dateien in dem Verzeichnis shoproot aus dem Modulpaket
       in das Hauptverzeichnis ihrer bestehenden modified eCommerce Shopsoftware Installation. Es werden dabei keine Dateien überschrieben!

    4. Wechseln sie in den Administrationsbereich und rufen sie den Menüpunkt Module -> System-Module auf.

    5. Markieren sie dort den Eintrag
       MITS_carrier_data_privacy - Einwilligung zur Datenweitergabe an Versanddienstleister © by Hetfield (MerZ IT-SerVice)
       und klicken sie dann auf der rechten Seite auf den Button Installieren. Das Modul wird nun komplett installiert.

    6. Konfigurieren sie nun das Modul nach ihren Wünschen. Die verschiedenen Einstellmöglichkeiten sind im Modul erklärt.

    7. Rufen sie den Menüpunkt Hilfsprogramme -> Paketdienstleister-Adressen auf.
       Dort können sie Adresseinträge anlegen, bearbeiten oder löschen und Adressen den jeweiligen Versandmodulen zuweisen.
